# Алгоритмы шифрования

В данном репозитории размещены 5 лабораторных работ по дисциплине "Информаицонная безопасность и защита информации", выполненные студентом группы `ОБ-02.03.02.01-41` *Колесниковым Ильёй*. Работы выполнены с использованием фреймворка Qt.

Код лабораторных работ можно посмотреть, выбрав ветку lab с соответствующим номером работы. Архив с собранными программами под платформу Windows 10(11) можно найти в разделе Realese. Описание программ и реализованных в них методах можно найти в readme.md на соответствующих ветках.

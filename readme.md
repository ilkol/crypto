# Лабораторная работа №6: Хэш функции

Программа реализует хэш функцию на основе блочного шифрования по ГОСТ-28147-89. Программа на вход принимает ключ (16 символа) и сообщение.
Хэш функция основана на схеме
H<sub>i</sub> = E<sub>H<sub>i-1</sub></sub>(M<sub>i</sub>) ⊕ M<sub>i</sub> ⊕ H<sub>i-1</sub>
H<sub>0</sub> = ключ шифрования
